# 전략 템플릿 명세 (MVP 3종)

MVP는 자유 전략 빌더 대신 **검증된 템플릿 3개**로 시작한다. 각 템플릿은 유저가 파라미터만 조정.
전략 조건 평가는 순수 함수로 구현 (입력: 캔들 배열 + 파라미터 → 출력: `LONG` / `SHORT` / `CLOSE` / `HOLD`).

> ⚠️ 모든 수치는 유저가 조정 가능한 파라미터. 아래는 기본값 제안이며, 플랫폼이 "이 값이 좋다"고 추천하지 않는다 (규제 가드레일).

---

## 공통 파라미터 (전 템플릿)

| 파라미터 | 설명 | 기본값(예시) |
|---|---|---|
| `symbol` | 거래 심볼 | BTCUSDT |
| `timeframe` | 캔들 주기 | 15m |
| `leverage` | 레버리지 배수 | 3x |
| `position_size` | 진입 규모 (잔고 %) | 10% |
| `stop_loss` | 손절 (%) | 2% |
| `take_profit` | 익절 (%) | 4% |

평가 시점: **캔들 종가 확정 시** (MVP는 tick 단위 아님).

---

## 1. 변동성 돌파 (Volatility Breakout)

전일 변동폭의 일정 비율을 당일 시가에 더한 목표가를 돌파하면 진입.

**고유 파라미터**
| 파라미터 | 설명 | 기본값 |
|---|---|---|
| `k` | 변동폭 계수 | 0.5 |

**로직**
- 목표가 = 당일(또는 현재 봉 기준 기간) 시가 + (전 구간 고가 − 전 구간 저가) × `k`
- 현재가가 목표가 상향 돌파 → `LONG`
- 손절/익절 도달 또는 구간 종료 → `CLOSE`

**엣지케이스:** 첫 구간(전일 데이터 없음)은 HOLD. `k`가 0이면 시가 즉시 돌파 → 방지 검증.

---

## 2. 이평선 교차 (MA Crossover)

단기 이동평균이 장기 이동평균을 교차하는 시점에 진입/청산.

**고유 파라미터**
| 파라미터 | 설명 | 기본값 |
|---|---|---|
| `fast_period` | 단기 MA 기간 | 7 |
| `slow_period` | 장기 MA 기간 | 25 |
| `ma_type` | SMA / EMA | EMA |

**로직**
- 단기 MA가 장기 MA를 **상향 교차** (골든크로스) → `LONG`
- 단기 MA가 장기 MA를 **하향 교차** (데드크로스) → `SHORT` (또는 롱 청산, 유저 설정)
- 손절/익절 우선 적용

**엣지케이스:** `fast_period >= slow_period` 이면 유효하지 않음 → UI에서 막기. 캔들 수가 `slow_period` 미만이면 HOLD.

---

## 3. RSI 과매수/과매도

RSI 지표가 과매도 구간에서 반등, 과매수 구간에서 하락 시 진입.

**고유 파라미터**
| 파라미터 | 설명 | 기본값 |
|---|---|---|
| `rsi_period` | RSI 기간 | 14 |
| `oversold` | 과매도 임계 | 30 |
| `overbought` | 과매수 임계 | 70 |

**로직**
- RSI가 `oversold` 아래로 갔다가 다시 위로 돌파 → `LONG`
- RSI가 `overbought` 위로 갔다가 다시 아래로 돌파 → `SHORT` (또는 롱 청산)
- 손절/익절 우선 적용

**엣지케이스:** `oversold >= overbought` 이면 유효하지 않음. 캔들 수가 `rsi_period + 1` 미만이면 HOLD.

## 구현 노트

- 세 전략 모두 **동일한 인터페이스**로: `evaluate(candles: Candle[], params: P): Signal`
- 손절/익절은 전략 시그널과 별개로 포지션 보유 중 **매 평가마다** 우선 체크 (전략 신호보다 리스크 관리가 먼저).
- 지표 계산(MA, RSI)은 검증된 로직으로. 직접 구현 시 반드시 단위 테스트 (알려진 입력→기대 출력).
